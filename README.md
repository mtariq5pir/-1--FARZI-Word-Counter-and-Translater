
   # Farzi Word Counter and Translator

## خصوصیات (Features)
* الفاظ اور حروف کی لائیو گنتی۔
* مختلف زبانوں میں ترجمہ کرنے کی صلاحیت۔
* خوبصورت اور آسان UI۔
